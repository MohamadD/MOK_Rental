```console
> git clone https://github.com/okarem/MOK_Rental
> cd MOK_Rental
> npm i
> google-chrome index.html
```
Install tape and tap-spec

```console
> npm i tape -D
> npm i tap-spec -D
```
Run the test command

```console
> npm test
```
